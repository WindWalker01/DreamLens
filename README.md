# DreamLens — HomeCredit

This repository contains the DreamLens project. The `HomeCredit` frontend prototype lives under `frontend/HomeCredit/` and is part of the DreamLens demo.

## About HomeCredit

`HomeCredit` is a frontend prototype included in this repository used to showcase the DreamLens project's UI and interactions for the HomeCredit dataset demo. It contains a Vite-based frontend with the core UI located in `frontend/HomeCredit/src`.

## Quick Start

Backend (Python):

1. Create and activate a virtual environment (PowerShell):

   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate
   pip install -r backend/requirements.txt
   python backend/main.py
   ```

Frontend (HomeCredit Vite app):

1. Install dependencies and run the dev server:

   ```powershell
   cd frontend/HomeCredit
   npm install
   npm run dev
   ```

## Project Structure (selected)

- `backend/` — Python backend (API and demo server).
- `frontend/HomeCredit/` — Vite frontend prototype for HomeCredit.
